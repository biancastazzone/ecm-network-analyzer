# ECM Network Analyzer

Bioinformatics pipeline to analyze gene networks involved in extracellular matrix remodeling using transcriptomic data.

## Features
- Differential expression analysis (DESeq2 / limma)
- Functional enrichment (GO, KEGG)
- Protein-protein interaction networks (STRING)
- Hub gene identification

## Goal
This project is being developed as a proposal for Google Summer of Code 2026 with EMBL-EBI.
